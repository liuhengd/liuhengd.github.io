安装配置hexo到Githob

（一）本地配置Hexo

1. 安装node-v7.7.1-x86.msi

2. 安装Git-2.12.0-32-bit

3. 打开http://npm.taobao.org/，设置npm镜像为淘宝

4. 安装完毕后，打开终端（windows->cmd+Enter回车键），检验是否安装正确。

5. 输入上面taobao 网页使用说明的命令

6. 继续输入命令，安装hexo： cnpm install–g hexo -cli

7. 新建文件夹MD到D盘根目录，复制地址D：/MD

8. 继续输入命令：cd /d D：/MD

9. 继续输入命令：D：/MD>hexoinit hexo ，等待系统完成所有配置

10. 进入该目录再执行cnpm install等待安装完成即可

11. 最后输入命令，浏览本地站点：D：/MD>hexohexo server

12. 根据最后一行打开网址http://localhost:4000/，界面如下图：即Hexo的默认页面

（二）在Github**上部署hexo**

1、 在Github上新建repository，名称必须是用户名.github.io，我的是liuhengd.github.io

2、 修改配置文件_config.yml的deploy部分，注意空格

1） 配置site部分

2） 安装 hexo-deployer-git

3） 配置git全局邮箱与名字

4） 生成博客

5） 部署博客

（三）添加新文章

1）生成新文章

2）用编辑器修改文章：myhexo\source_posts\newblog.md

3）生成并发布到Github
